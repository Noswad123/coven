# Example project context

- The starter goal is an interactive Coven demo agent.
- The completed app should run locally with `python3 output/demo.py`.
- It should read this workspace's goal, dashboard, tasks, checkpoints, work docs, and output docs, then answer user questions about the collaboration.
- It should explain who did what, why decisions were made, which checkpoints were approved, and what QA verified.
- It should work without setup via deterministic answers and may optionally use a free/local model such as Ollama when configured.
- Treat this as a small end-to-end demonstration of planning, implementation, UI polish, QA, and user checkpoint approval.
